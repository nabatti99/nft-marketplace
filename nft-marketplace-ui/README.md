# Smart PPA Frontend

This project is a demo UI for SmartPPA contract using **Vite** for fast React development with TypeScript, TailwindCSS, and other modern tools for building, linting, formatting, and testing.

## Usage

### Prerequisites
Make sure you have the following installed on your machine:

- [Node.js](https://nodejs.org/) should be installed by [nvm](https://github.com/nvm-sh/nvm) (Node Version Manager).

### Installation
Enable `Yarn` by running the following command in the project root directory:

```shell
corepack enable
```

To install the project dependencies, run the following command in the project root directory:

```shell
yarn install
```

### Development
To start the development server, run the following command in the project root directory:

```shell
yarn dev
```
