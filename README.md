# NFT Marketplace

An end-to-end NFT marketplace project with:

- Solidity smart contracts (Hardhat)
- NFT Marketplace UI (React + ethers v6 + MetaMask + HeroUI)

The marketplace contract supports listing, canceling, and buying ERC721 NFTs using XSGD token payments.

## Tech Stack

- Node.js
- Yarn (via Corepack)
- Hardhat 3 + Solidity
- React + Vite
- ethers v6
- MetaMask
- HeroUI

## Repository Layout

- `/nft-marketplace-contract` - Solidity contracts, Hardhat config, deployment and helper scripts, tests, and Ignition modules
- `/nft-marketplace-ui` - React frontend for the marketplace dApp
